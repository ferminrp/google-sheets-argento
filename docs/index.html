Qué hace

Un script de Apps Script que agrega funciones personalizadas a tu Google Sheet. Escribís una fórmula y la celda se llena con datos financieros de Argentina.

Dólar y BCRA

Blue, oficial, MEP, CCL, tarjeta, histórico y variables del BCRA.

=DOLAR("blue")

Mercado argentino

Acciones, bonos, letras, opciones y obligaciones negociables.

=ACCIONES("YPF";"c")

CEDEARs y USA

Precios, ratios y paneles de CEDEARs y acciones de EE.UU.

=CEDEAR("AAPL";"c")

Indicadores

Inflación mensual, UVA y riesgo país, con fecha opcional.

=INFLACION()

Crypto

Precios spot y comparador por exchange (CriptoYa), más APYs.

=CRIPTOYA("BTC";"ARS")

Inversiones

Plazos fijos, FCI y cálculo de cauciones tomadora/colocadora.

=PLAZOFIJO("Nacion")

Cómo instalarlo

No hace falta Node, clasp ni extensiones de Chrome. Solo copiar un archivo en Apps Script.

  1. Abrí tu Google Sheet

    Creá una planilla nueva o usá una existente donde quieras las fórmulas.

  2. Entrá a Apps Script

    Menú Extensiones → Apps Script. Se abre el editor de Google.

  3. Pegá el código completo

    Borrá el contenido por defecto del editor y pegá el archivo all-in-one.js del repositorio (incluye todas las funciones).

    Abrir all-in-one.js (raw) Ver en GitHub

  4. Guardá el proyecto

    Ctrl+S o ⌘+S. Podés renombrar el proyecto a “Sheets Argento” si querés.

  5. Volvé a la hoja y probá

    La primera vez que uses una fórmula, Google pedirá autorizar el script (acceso a servicios externos). Aceptá y listo. 

    =DOLAR("blue")
Tips de instalación
  • En Sheets con locale argentino, el separador de argumentos suele ser ; (no coma).
  • No necesitás instalar nada en la compu: todo corre en la nube de Google.
  • Para actualizar, volvé a pegar el all-in-one.js más reciente del repo.

Cómo usarlo

Escribí las fórmulas en cualquier celda, igual que =SUMA(). Algunas funciones devuelven un número; otras, tablas completas.

Fórmula Qué devuelve
=DOLAR("blue") Venta del dólar blue (default)
=DOLAR("mep";"venta") MEP (alias de bolsa)
=DOLAR("ccl";"promedio") CCL promedio (alias de contadoconliqui)
=DOLARHISTORICO("blue";"15/01/2024") Blue en una fecha (también acepta YYYY-MM-DD)
=ACCIONES("GGAL";"c") Precio de Galicia
=ACCIONESHISTORICO("GGAL";"2024-01-15") Cierre histórico de Galicia (OHLC data912)
=ACCIONESLISTA() Tabla con todo el panel de acciones AR
=BONOS("AL30";"pct_change") Variación diaria del AL30
=BONOSHISTORICO("AL30";"2024-06-01";"dr") Retorno diario histórico del AL30
=CEDEAR("AAPL";"c") Precio del CEDEAR de Apple
=CEDEARHISTORICO("AAPL";"15/01/2024") Cierre histórico del CEDEAR de Apple
=CEDEAR("AAPL";"ratio") Ratio CEDEAR / acción
=INFLACION() Último dato de inflación mensual
=UVA() Valor actual del UVA
=RIESGOPAIS() Riesgo país más reciente
=BCRA(1) Reservas internacionales (variable BCRA id 1)
=CRIPTOYA("BTC";"ARS") Mejor precio BTC/ARS entre exchanges
=CRYPTO("BTC") Spot BTC en USD (Coinbase)
=PLAZOFIJO() Mejor TNA de plazo fijo (clientes)
=FCI("mercadoDinero";"Balanz Money Market USD - Clase A") Valor cuotaparte del FCI
=CAUCIONCOLOCADORA(30; 40%; 1000000) Resultado de una caución colocadora
Atributos de paneles de mercado
  • c precio · v volumen · pct_change variación %
  • px_bid / px_ask bid/ask · q_bid / q_ask cantidades · q_op operaciones
  • Vale para acciones, bonos, letras, opciones, ONs, CEDEARs y USA stocks

Cuándo se actualizan los datos

Google Sheets recalcula al abrir la hoja o al editar celdas. Podés forzar con Ctrl+R (Windows) o ⌘+R (Mac), o en Archivo → Configuración → Cálculo. Las cotizaciones live usan cache de ~60 s en Apps Script para no saturar las APIs.

API / datos abiertos

Miscelánea de la página: JSON estático servido por GitHub Pages. Pensado para agentes, scripts y quien no quiera pelearse con el PDF de BYMA.

CEDEARs + ratios

Listado con ticker, nombre, mercado del subyacente y ratio de conversión. Ideal para alimentar un agente o un script sin scrapear el PDF de BYMA.

GET api/cedears.json

Content-Type: application/json

Endpoint

https://ferminrp.github.io/google-sheets-argento/api/cedears.json

Forma de la respuesta

{
  "schema_version": 1,
  "name": "cedears",
  "count": 428,
  "fields": {
    "Cedears": "Ticker del CEDEAR en BYMA",
    "Name": "Nombre de la empresa / instrumento subyacente",
    "Market": "Mercado del subyacente (ej. NYSE, NASDAQ, B3)",
    "Ratio": "String de conversión…"
  },
  "items": [
    { "Cedears": "AAPL", "Name": "Apple Inc.", "Market": "NASDAQ", "Ratio": "20" },
    { "Cedears": "ABEV", "Name": "Ambev S.A.", "Market": "NASDAQ", "Ratio": "1:3" }
  ]
}
Cómo leer el ratio
  • "20" → 20 CEDEARs = 1 acción subyacente (ej. AAPL)
  • "1:3" → razón 1:3 (formato alternativo en algunos tickers)
  • El valor se expone sin normalizar; parsealo según el formato del ticker
  • Fuente canónica en el repo: data/cedears.json (array plano). El endpoint de Pages agrega metadata para agentes.

Ejemplos

# curl
curl -s https://ferminrp.github.io/google-sheets-argento/api/cedears.json | jq '.items[] | select(.Cedears=="AAPL")'

# JavaScript
const res = await fetch('https://ferminrp.github.io/google-sheets-argento/api/cedears.json');
const { items } = await res.json();
const aapl = items.find((c) => c.Cedears === 'AAPL');

Es un snapshot versionado en el repo, no una API de cotizaciones en vivo. Los precios siguen viniendo de data912 vía las fórmulas de Sheets. CORS: GET público (ideal para fetch desde scripts y agentes).

Preguntas frecuentes

Cosas que suelen aparecer al instalar o usar las fórmulas.

¿Es gratis?

Sí. El código es open source. Las APIs que usa también son públicas (sujetas a los términos de cada proveedor). Podés invitar un cafecito si te resulta útil.

¿Funciona sin internet?

No. Cada recálculo de fórmula hace (o lee del cache) un request a las APIs. Si no hay red o la API cae, la celda muestra un error.

¿Por qué a veces el valor no cambia al toque?

Hay dos capas: el cache de Apps Script (~60 s en cotizaciones live) y el motor de recálculo de Google Sheets. Forzá con Ctrl/⌘+R o editá una celda.

¿Cómo actualizo el script?

Abrí de nuevo Apps Script, reemplazá el contenido por el all-in-one.js actual del repo y guardá.

¿Puedo usar “mep” y “ccl” en vez de bolsa / contadoconliqui?

Sí. mep es alias de bolsa y ccl / contado de contadoconliqui.

Mi fórmula usa comas y no funciona

En muchas cuentas de Google en Argentina el separador de argumentos es punto y coma: =DOLAR("blue";"venta"). Si tu hoja usa coma, probá con el locale de la planilla.

¿Puedo contribuir o reportar un bug?

Sí: abrí un issue o un PR en GitHub. Para desarrollo local: npm install, npm run build, npm test.
